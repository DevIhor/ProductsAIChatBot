# ProductsAIChatBot
A demo chat bot that has information about the list of products.

## Setup
- You need to create `.env` file with filled OPENAI_API_KEY;
```bash
cp .env.example .env
```
- Install pipenv (python - 3.10);
- Create virtual environment using pipenv:
```bash
pipenv sync --dev
```
- Run bot with:
```bash
python main.py
```
